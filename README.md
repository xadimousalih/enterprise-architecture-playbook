# 🌍 Overview

Enterprise Architecture Playbook is an open-source knowledge platform designed for:

- Enterprise Architects
- Solution Architects
- Cloud Architects
- Software Architects
- Platform Engineers
- Technical Leaders

It provides practical guidance, reusable architecture patterns, reference architectures, governance models, Architecture Decision Records (ADR), cloud blueprints, and architecture review templates.

Unlike traditional frameworks, this project focuses on **execution**, **modern architectures**, and **real-world implementation**.

# 🚀 Quick Start

Clone the repository

```bash
git clone https://github.com/xadimousalih/enterprise-architecture-playbook.git
```

Install dependencies

```bash
pip install mkdocs-material
```

Run locally

```bash
mkdocs serve
```

Open

```text
http://127.0.0.1:8000
```
