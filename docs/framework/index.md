# Enterprise Architecture Framework

## Why another framework?

Traditional Enterprise Architecture frameworks were designed before Cloud Native, Kubernetes, APIs, Platform Engineering and Artificial Intelligence.

The Enterprise Architecture Playbook introduces a pragmatic framework that complements TOGAF by focusing on execution.

---

## Core Pillars

- Business Alignment
- Platform Thinking
- Cloud Native
- API First
- Event Driven
- Security by Design
- Observability by Default
- AI Ready
- Automation First
